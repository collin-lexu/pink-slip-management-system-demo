# Pink Slip Management System
A web application built for Anna's Alterations to digitize and analyze paper-based alteration order records. The system supports data entry, CSV/Excel import, and SQL-driven analysis of customer behavior, revenue trends, and seasonal demand patterns.

This demo uses sample data spanning 2021–2023, designed to simulate realistic order volume, seasonal trends, and customer behavior patterns similar to what a small alterations shop would generate. All names, phone numbers, and customer details are randomly generated. Any resemblance to real individuals is purely coincidental.
